Description

I built a powerful chatbot using Google APIs that understands both text and images. With Google Cloud Natural Language API, it processes human language, interpreting queries and sentiments for smooth, natural conversations. It also uses Google Cloud Vision API to analyze images, reading text from photos with OCR and recognizing objects and landmarks. This dual capability allows users to interact with the bot through both text and images, making it versatile and intelligent. The chatbot adapts and improves over time, providing a more responsive and engaging experience.
